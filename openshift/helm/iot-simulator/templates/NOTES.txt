You will need to create a new endpoint config (e.g. for endpoint "ep1"):

    oc create secret generic ep1 \
      --from-file=ca.crt=certs/ca.crt \
      --from-literal=endpoint.username=<foo> \
      --from-literal=endpoint.password=<bar>

And also create a new config map:

    oc create configmap ep1 \
      --from-literal=endpoint.host=… \
      --from-literal=endpoint.port=443… \
      --from-literal=deviceRegistry.url=https://… \
      --from-literal=httpAdapter.url=https://… \
      --from-literal=mqttAdapter.host=… \
      --from-literal=mqttAdapter.port=443…

And then create new "SimulatorConsumer" and "SimulatorProducer" instances.
